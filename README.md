inradius/ng-drupal-starter
====

This project is just my attempt to built an Angular 4+ SPA that uses Drupal 8 as a content source REST api.

This is a work in progress.

### Installation
* `$ yarn install`
* `$ yarn build:dll`
* `$ yarn start`
* Go to http://localhost:8080
